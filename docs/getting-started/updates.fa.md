# به‌روزرسانی برنامه

Mycel به طور فعال توسعه می‌یابد. به‌روزرسانی‌ها رفع باگ، بهبود عملکرد، و ویژگی‌های جدید می‌آورند.

## بررسی به‌روزرسانی‌ها

### بررسی دستی

۱. به [github.com/ethene/mycel-releases/releases](https://github.com/ethene/mycel-releases/releases) بروید
۲. آخرین نسخه را با نسخه نصب‌شده خود مقایسه کنید
۳. اگر به‌روزرسانی هست، APK جدید را دانلود و نصب کنید

نسخه فعلی شما در **تنظیمات** > **درباره** نشان داده می‌شود.

### به‌روزرسانی خودکار با Obtainium

[Obtainium](https://github.com/ImranR98/Obtainium) یک برنامه رایگان است که نسخه‌های GitHub را نظارت می‌کند و از به‌روزرسانی‌ها خبر می‌دهد.

**راه‌اندازی:**

۱. Obtainium را از [F-Droid](https://f-droid.org/packages/dev.imranr.obtainium.fdroid/) یا [GitHub](https://github.com/ImranR98/Obtainium/releases) نصب کنید
۲. Obtainium را باز کنید و روی **+** (افزودن برنامه) بزنید
۳. وارد کنید: `https://github.com/ethene/mycel-releases`
۴. روی **افزودن** بزنید

Obtainium حالا نسخه‌های جدید را بررسی می‌کند و وقتی به‌روزرسانی‌ها در دسترس است به شما اطلاع می‌دهد.

## نصب به‌روزرسانی‌ها

به‌روزرسانی‌ها روی برنامه موجود نصب می‌شوند - **داده‌های شما حفظ می‌شود**.

۱. APK جدید را دانلود کنید
۲. آن را باز کنید و روی **نصب** بزنید
۳. به‌روزرسانی جایگزین نسخه قدیمی می‌شود
۴. مخاطبین، پیام‌ها، و تنظیمات شما دست‌نخورده می‌مانند

!!! note "همان کلید امضا"
    به‌روزرسانی‌ها باید با همان کلید نسخه نصب‌شده شما امضا شده باشند. اگر «امضاها مطابقت ندارند» دیدید، ممکن است نسخه غیررسمی نصب داشته باشید.

## شماره‌گذاری نسخه

Mycel از شماره نسخه‌هایی مانند `2.0.720` استفاده می‌کند:

- **2** - نسخه اصلی
- **0** - نسخه فرعی
- **720** - پچ/شماره ساخت

اعداد بالاتر = نسخه‌های جدیدتر.

## یادداشت‌های انتشار

هر انتشار شامل یادداشت‌هایی است که تغییرات را توضیح می‌دهد. [صفحه انتشارات](https://github.com/ethene/mycel-releases/releases) را برای جزئیات هر نسخه ببینید.

## بازگشت (اگر نیاز شد)

اگر یک به‌روزرسانی مشکل ایجاد کرد:

۱. نسخه فعلی را حذف کنید (این داده‌های شما را پاک می‌کند)
۲. APK قدیمی‌تر را از [انتشارات](https://github.com/ethene/mycel-releases/releases) دانلود کنید
۳. نسخه قدیمی‌تر را نصب کنید

!!! warning "از دست رفتن داده هنگام حذف"
    حذف نصب، پیام‌ها و هویت شما را پاک می‌کند. هر داده مهمی را قبل از حذف صادر کنید.
