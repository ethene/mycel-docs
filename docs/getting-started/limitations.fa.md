# آنچه Mycel می‌تواند (و نمی‌تواند) انجام دهد

درک اینکه Mycel برای چه طراحی شده کمک می‌کند انتظارات درستی داشته باشید.

## چه چیزی الان کار می‌کند

| ویژگی | توضیح |
|-------|-------|
| **ارسال پیام بدون اینترنت** | از بلوتوث و وای‌فای بین گوشی‌های نزدیک استفاده می‌کند |
| **رمزنگاری سرتاسری** | فقط شما و گیرنده می‌توانید پیام‌ها را بخوانند |
| **بدون نیاز به شماره تلفن** | سیستم هویت حافظ حریم خصوصی |
| **گروه‌های خصوصی** | پیام‌رسانی گروهی رمزنگاری‌شده فقط با دعوت |
| **کار در هنگام قطعی** | طراحی شده برای سناریوهای بدون اینترنت |
| **تحویل در پس‌زمینه** | وقتی برنامه باز نیست کار می‌کند |

## چه چیزی زمان می‌برد

Mycel یک **شبکه با تحمل تأخیر** است - تحویل می‌تواند زمان ببرد. این عادی است، نه خطا.

| سناریو | زمان مورد انتظار |
|--------|-----------------|
| یک اتاق/ساختمان | ثانیه تا دقیقه |
| یک محله | دقیقه تا ساعت |
| یک شهر | ساعت‌ها |
| شهرهای مختلف | ساعت‌ها تا روزها |
| مناطق دورافتاده | روزها یا بیشتر |

!!! info "چرا این‌قدر کند؟"
    پیام‌ها از طریق **افراد** سفر می‌کنند، نه کابل‌ها. هر پرش نیاز به نزدیکی فیزیکی بین کاربران Mycel دارد.

## آنچه Mycel نمی‌تواند انجام دهد

این‌ها **باگ نیستند** - ویژگی‌های اساسی شبکه مش هستند:

| محدودیت | دلیل |
|---------|------|
| **تضمین تحویل** | اگر زنجیره‌ای از کاربران وجود نداشته باشد، پیام‌ها صبر می‌کنند |
| **پیام‌رسانی فوری** | ذخیره-و-فوروارد زمان می‌برد |
| **کار روی همه گوشی‌ها** | نیاز به اندروید ۸.۰+ با Google Play Services دارد (ویژگی‌های کامل مش به اندروید ۱۰+ نیاز دارند) |
| **تحویل به آیفون** | برنامه iOS هنوز در دسترس نیست |

## نیازمندی‌های فنی

| نیازمندی | جزئیات |
|----------|--------|
| **نسخه اندروید** | اندروید ۸.۰ (API 26) یا بالاتر. ویژگی‌های کامل مش به اندروید ۱۰+ نیاز دارند |
| **Google Play Services** | برای اتصالات نزدیک لازم است |
| **بلوتوث** | باید فعال باشد |
| **بهینه‌سازی باتری** | باید برای Mycel غیرفعال باشد |

!!! warning "دستگاه‌های هواوی (بعد از ۲۰۱۹)"
    بیشتر دستگاه‌های هواوی تولید شده بعد از ۲۰۱۹ Google Play Services ندارند و نمی‌توانند Mycel را اجرا کنند.

## برای مواقع اضطراری نیست

!!! danger "مهم"
    **برای ارتباطات اضطراری به Mycel تکیه نکنید.**

    - تحویل می‌تواند ساعت‌ها یا روزها طول بکشد
    - تضمینی برای رسیدن پیام‌ها نیست
    - برای مواقع اضطراری، از خدمات اضطراری رسمی استفاده کنید

## مشکل شروع سرد

وقتی Mycel برای اولین بار در یک منطقه راه‌اندازی می‌شود:

- **حامل‌های کمی وجود دارند** - پیام‌ها بیشتر طول می‌کشند
- **پذیرندگان اولیه در حال ساختن شبکه هستند** - هر کاربر کمک می‌کند
- **صبر لازم است** - تحویل با رشد شبکه بهبود می‌یابد

هرچه افراد بیشتری بپیوندند، تحویل سریع‌تر و قابل اعتمادتر می‌شود.
