## GitHub

### Report Bugs

Found a bug? Report it on GitHub:

**[github.com/ethene/mycel-releases/issues](https://github.com/ethene/mycel-releases/issues)**

When reporting, please include:

- Your Android version
- Your device model
- Steps to reproduce the issue
- Screenshots if relevant

### Request Features

Have an idea for Mycel? Open a feature request on GitHub Issues.

## Before Asking for Help

To get the best help, please provide:

1. **What you tried** - Steps you took
2. **What happened** - The actual result
3. **What you expected** - The desired result
4. **Device info** - Android version, device model
5. **App version** - Found in Settings

## Limitations

Please understand that Mycel is:

- **Not for emergencies** - Delivery can take hours/days
- **Experimental** - We're still improving stability
- **Android-only** - iOS version is not yet available

For critical communications, use traditional methods.
