# User Guide

Everything you need to know about using Mycel day-to-day.

## Conversations

Mycel supports three types of conversations:

| Type | Who can read | Who can post | Best for |
|------|--------------|--------------|----------|
| [Direct Messages](messaging.md) | You and one contact | Both of you | Private 1:1 chats |
| [Groups](groups.md) | Group members | Any member | Team discussions |
| [Channels](channels.md) | Anyone subscribed | Channel owner only | Announcements, broadcasts |

There's also [Mesh Broadcast](mesh-broadcast.md) - a special local chat shared with everyone nearby.
