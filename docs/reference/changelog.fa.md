# تاریخچه تغییرات

همه تغییرات قابل توجه Mycel در اینجا مستند شده است.

## [2.1.1] - ۲۰۲۶-۰۱-۱۳

### رفع شده

- سیستم نسخه حالا از version.txt می‌خواند (منبع حقیقت واحد)
- تأیید امضای CRDT برای GROUP_UPDATE از طریق Nostr

## [2.1.0] - ۲۰۲۶-۰۱-۱۲

**نسخه انتشار نرم ایران**

### ویژگی‌ها

- بومی‌سازی کامل فارسی (fa-IR) - ۴۸۶ رشته ترجمه شده
- اشتراک دعوت دوزبانه (انگلیسی + فارسی)
- انتشار CRDT فقط از طریق Nostr برای عضویت گروه
- سیستم تحلیل خود-مدیریت با داشبورد فرضیه

### فنی

- انتشار موضوع CRDT از طریق Nostr
- تأیید امضا با استفاده از کلیدهای عمومی Ed25519 خام

---

## [2.0.720] - ۲۰۲۶-۰۱-۰۲

**اولین انتشار عمومی**

### ویژگی‌ها

- پیام‌های مستقیم رمزنگاری‌شده سرتاسری
- گروه‌های خصوصی با همگام‌سازی عضویت CRDT
- کانال‌های پخش
- مسیریابی مش چند پرش
- انتقال نزدیک (Google Nearby Connections)
- انتقال Nostr (رله اینترنتی)
- تأیید تحویل مبتنی بر ACK
- مسیریابی PRoPHET + پاشش-و-صبر
- خوشه‌بندی جغرافیایی H3

### فنی

- اندروید ۸.۰+ (API 26+)، ویژگی‌های کامل مش در اندروید ۱۰+
- Kotlin 1.9.24
- رابط کاربری Jetpack Compose
- پایگاه داده Room
- کوچک‌سازی R8 فعال
