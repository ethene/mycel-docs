# سیاست حریم خصوصی

**آخرین به‌روزرسانی:** ۱۳ دی ۱۴۰۴ (۲ ژانویه ۲۰۲۶)

## نمای کلی

Mycel یک برنامه پیام‌رسانی غیرمتمرکز و همتا به همتا است. این سیاست حریم خصوصی توضیح می‌دهد چه داده‌ای Mycel جمع‌آوری می‌کند، چگونه استفاده می‌شود، و حقوق شما.

**نسخه کوتاه:** Mycel هیچ داده شخصی جمع‌آوری نمی‌کند. پیام‌ها سرتاسری رمزنگاری می‌شوند. هیچ سروری اطلاعات شما را ذخیره نمی‌کند.

## داده‌ای که جمع‌آوری می‌کنیم

### داده‌ای که جمع‌آوری نمی‌کنیم

- اطلاعات شخصی (نام، ایمیل، شماره تلفن)
- داده‌های موقعیت
- تحلیل استفاده
- محتوای پیام
- لیست مخاطبین
- شناسه‌های دستگاه ارسال شده به هیچ سروری

### داده ذخیره شده روی دستگاه شما

Mycel موارد زیر را **فقط به صورت محلی روی دستگاه شما** ذخیره می‌کند:

| داده | هدف |
|------|-----|
| کلیدهای رمزنگاری | هویت شما (هرگز دستگاه را ترک نمی‌کند) |
| پیام‌ها | گفتگوهای شما (رمزنگاری‌شده) |
| مخاطبین | افرادی که با آن‌ها ارتباط برقرار می‌کنید |
| جداول مسیریابی | برای بهینه‌سازی تحویل پیام |

این داده در یک پایگاه داده محلی ذخیره می‌شود و به هیچ سروری منتقل نمی‌شود.

## نحوه کار پیام‌ها

### رله همتا به همتا

پیام‌ها از طریق دستگاه‌های کاربران Mycel دیگر برای رسیدن به مقصد سفر می‌کنند. هنگام رله کردن:

- دستگاه‌های دیگر به طور موقت پیام رمزنگاری‌شده شما را ذخیره می‌کنند
- آن‌ها نمی‌توانند محتوا را رمزگشایی یا بخوانند
- می‌توانند شناسه‌های مقصد را ببینند (برای مسیریابی پیام)
- پیام‌ها پس از تحویل یا انقضا حذف می‌شوند

### رمزنگاری سرتاسری

همه پیام‌ها روی دستگاه شما قبل از انتقال رمزنگاری می‌شوند. فقط گیرنده مورد نظر می‌تواند آن‌ها را رمزگشایی کند. حتی دستگاه‌های رله نمی‌توانند پیام‌های شما را بخوانند.

## خدمات شخص ثالث

### Nostr (اختیاری)

اگر انتقال Nostr را فعال کنید:

- پیام‌ها از طریق سرورهای عمومی Nostr رله می‌شوند
- این سرورها ممکن است ابرداده اتصال را ثبت کنند
- محتوای پیام همچنان رمزنگاری‌شده باقی می‌ماند
- می‌توانید Nostr را برای حداکثر حریم خصوصی غیرفعال کنید

### Google Nearby Connections

Mycel از API اتصالات نزدیک گوگل برای کشف دستگاه محلی استفاده می‌کند. این API به صورت محلی کار می‌کند و داده‌ای به سرورهای گوگل ارسال نمی‌کند.

## حقوق شما

می‌توانید:

- **داده‌های خود را حذف کنید** با حذف نصب برنامه
- **Nostr را غیرفعال کنید** برای عملیات فقط آفلاین
- **داده‌های ذخیره‌شده را مشاهده کنید** در دایرکتوری داده برنامه
- **پیام‌های خود را صادر کنید** (ویژگی برنامه‌ریزی شده)

## نگهداری داده

- پیام‌ها تا تحویل یا انقضا (TTL) ذخیره می‌شوند
- داده محلی تا حذف نصب برنامه باقی می‌ماند
- هیچ داده‌ای روی هیچ سروری نگهداری نمی‌شود (سروری وجود ندارد)

## امنیت

ما پیاده‌سازی می‌کنیم:

- رمزنگاری سرتاسری (X25519 + AES-GCM)
- امضاهای دیجیتال (Ed25519)
- بدون نقاط مرکزی خرابی
- کد متن‌باز برای شفافیت

## حریم خصوصی کودکان

Mycel برای کودکان زیر ۱۳ سال طراحی نشده است. ما عمداً اطلاعاتی از کودکان جمع‌آوری نمی‌کنیم.

## تغییرات در این سیاست

ممکن است این سیاست را به‌روزرسانی کنیم. تغییرات در تاریخچه تغییرات و این صفحه ذکر می‌شوند.
